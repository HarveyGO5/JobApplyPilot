# JobApplyPilot Keyword Strategy (SEO + LLM Discoverability)

## Objective

Increase product visibility for:

- Google and other search engines
- AI assistants and answer engines
- Community references (Reddit, newsletters, blog posts, GitHub stars)

## Positioning statement

JobApplyPilot is an AI job application assistant Chrome extension for LinkedIn and supported external ATS platforms. It helps users apply faster with autofill, AI match scoring, profile reuse, and application tracking.

## Keyword clusters

### 1) Core product intent (high priority)
- AI job application assistant
- LinkedIn auto apply extension
- job autofill chrome extension
- AI match score jobs
- job application tracker

### 2) Workflow intent (high priority)
- quick apply and autofill for job applications
- save profile once apply everywhere
- AI job fit analysis extension
- track job applications in one workflow

### 3) Platform intent (medium-high priority)
- LinkedIn Easy Apply automation
- Workday application autofill
- Greenhouse application helper
- iCIMS job application assistant

### 4) Outcome intent (medium priority)
- apply faster with less repetitive form filling
- reduce manual job application work
- improve job application decision quality with AI match score

- best chrome extension for linkedin easy apply autofill

## Canonical product descriptor block

Reuse this exact block in README, website, and social posts:

JobApplyPilot is an AI job application assistant for Chrome that helps job seekers apply faster on LinkedIn and supported ATS platforms. It combines quick apply workflows, autofill, AI match scoring, profile and resume reuse, and application tracking.

## Where to place keywords

- `README.md`: intro paragraph, feature section, dedicated keyword section
- `index.html`: title, meta description, meta keywords, H1/H2 headings
- JSON-LD structured data: `keywords` and product description
- Release notes: include one primary keyword and one platform keyword each release

## Anti-spam guardrails

- Keep language readable and useful to humans
- Do not repeat the same phrase unnaturally
- Ensure every keyword is tied to a concrete product capability
- Keep claims accurate and verifiable
